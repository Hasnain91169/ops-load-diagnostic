Operations Load Diagnostic

This repository is a lightweight, read-only diagnostic tool for logistics operations teams. It classifies inbound operational communications and produces a static workload report.

How to Run

Install and run locally with: pip install -e . then ops-diagnostic --mode csv --input examples/sample_inbound.csv --format both.

Output

Reports are generated in the output/ folder as Markdown/HTML plus a summary JSON.

Input Modes

CSV, plain text batch, and optional read-only IMAP ingestion are supported.
